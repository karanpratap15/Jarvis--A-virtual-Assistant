# Jarvis--A-virtual-Assistant
Jarvis is a minimalist virtual assistant built in Python that utilizes Google's advanced **Gemini API** (`gemini-2.5-flash`) to answer queries and process conversational requests efficiently. 
<br><br>
Designed with modern development best practices, this project features a secure setup that leverages system environment variables to handle sensitive API credentials, making it entirely safe for open-source sharing on GitHub.

---

## 🚀 Features

* **Powered by Gemini:** Uses the latest `google-genai` SDK for ultra-fast, intelligent streaming responses.
<br>
* **Production Secure:** Zero hardcoded API keys. Authentication is handled globally via system-level environment variables.
<br>
* **Lightweight Architecture:** Minimal dependencies and a clean, scannable project structure.

---

## 🛠️ Tech Stack & Prerequisites

* **Language:** Python 3.10+
<br>
* **LLM Engine:** Google Gemini (`gemini-2.5-flash`)
<br>
* **SDK:** `google-genai`
<br>
* **Environment:** Windows PowerShell / Virtual Environment (`.venv`)
